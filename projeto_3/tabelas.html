O elemento table (tabela)

O elemeto table (tabela) representa dados tubulares ou seja, informações que são representados em linhas e colunas.

Exemplo

Tabela simples com cabeçalho

Primeiro nome Sobrenome
Washingon Paiva
André Greroski

Elementos da tabela

  • table - tabela;
  • th (table header) - cabeçalho da coluna;
  • tr (table row) - linha de tabela;
  • td (table data) - valor/conteúdo de célula;
